## 풀잎 진행방식

- 내용 정리
    - 코드는 눈으로 보고 넘기기
    - 각자 읽은 부분 중, 다음에 해당하는 내용들을 정리
        - 중요하게 생각하는 것
        - 나에게 특히 영감을 준 부분
        - 의문점이 드는 것
        - (선택) 코드에 적용해 본다면?
- 발표
    - 발표자는 정리된 내용을 토대로 발표
    - 발표를 들은 사람들 중 랜덤으로 한명을 선정하여 발표자의 내용을 간단히 요약해서 다시 발표
